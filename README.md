# Weather-App

Build a simple weather forecasting application using Node.js, using different API's.


Geocode API: OpenCage Geocoder API

Geocode API: Dark Sky API


## Steps to Setup

1. Install dependencies

```bash
npm install
```

2. Run Command

```bash
node index.js -a " { Your Address } "
```
